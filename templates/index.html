Digital Twin Components

Workspace

The Workspace is your digital twin's home in AWS IoT TwinMaker. It includes:

  • Unique workspace ID
  • S3 bucket for assets
  • IAM role for permissions
  • Description and metadata

Component Types

Component Types define the capabilities and properties of your twin's elements:

  • Property definitions
  • Data types
  • Time series properties
  • Default values

Entities

Entities represent physical objects in your factory:

  • Machines and equipment
  • Sensors and devices
  • Factory areas
  • Implemented component types

Assets

Assets connect to AWS IoT SiteWise assets:

  • External asset links
  • Asset hierarchies
  • Properties and telemetry
  • Metadata and tags

Scene

Scenes provide 3D visualization of your factory:

  • 3D models and layouts
  • Scene components
  • Camera positions
  • S3 storage location

Connections

Connections define relationships between entities:

  • Physical connections
  • Logical relationships
  • Hierarchical structures
  • Data flow paths
